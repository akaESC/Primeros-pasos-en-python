# Mis primeros pasos en Python
Repositorio personal para guardar mis avances, scripts de lógica y ejercicios de práctica.

## Scripts incluidos y ordenados:
* **acceso.py**: Gestor de entrada básico.
* **barberia.py**: Gestión de colas y clientes.
* **calculadora_propina.py**: Cálculo rápido de servicios.
* **entrada_vip.py**: Gestor de entrada básico.
* **gestor_gastos.py**: Control de presupuesto personal.
* **gestor_ingresos.py**: Gestor basico de ingresos en cuenta de banco.
* **piezas.py**: Enlistado básico sobre piezas (añadiendo componentes olvidados).
* **presupuesto.py**: Gestor básico sobre presupuestos.
* **repaso.py**: Gestor de entrada para Usuarios básico (repaso).
